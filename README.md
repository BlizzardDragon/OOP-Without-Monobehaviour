# Space Arcade

### Описание
Небольшая аркада, написанная в классической ООП парадигме. В проекте используется самописный DI контейнер и система модулей, которая позволяет отказаться от большинства Monobehaviour.

### Особенности
- ООП
- Кастомный DI
- Без монобехов

### Управление
- A, D 
- Space
